House Size Estimation Model

|Personal Project|

This project applies a machine learning model to predict missing house_size values in a large real estate dataset containing over 2,200,000 properties. A LightGBM regression pipeline was trained and achieved a typical error margin of ±300 sq ft. The workflow includes outlier detection, feature preprocessing, and a structured validation process using MAE, RMSE, and R² metrics. All missing house sizes were successfully imputed, and the final dataset includes metadata flags for predicted values and outlier classification. This project demonstrates scalable data cleaning and predictive modeling in a real-world context.
